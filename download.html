Install MathGPT

Follow these simple steps to install MathGPT from GitHub

1 Download from GitHub

Visit the MathGPT GitHub repository, click on the "Code" button and download the extension as a ZIP file. Or click on the Download Zip button to download directly

Download ZIP

2 Enable Developer Mode

Open Chrome and navigate to chrome://extensions. Toggle the "Developer mode" on, in the top-right corner.

3 Extract the ZIP File

Extract the downloaded ZIP file to a location on your computer where you want to keep the extension files.

4 Load Unpacked Extension

Go back on chrome://extensions. Click "Load unpacked" and select the folder we just extracted. Make sure to select the main extracted folder.

5 Start Using MathGPT

Navigate to ChatGPT and start using mathematical symbols! The MathGPT panel will appear on the right side of your screen.
